English-to-Python Converter

Tackled the challenge of bridging natural language and programming for beginners. Built a web application leveraging Python, Flask, spaCy, and Syntax-Directed Translation (SDT) to transform plain English algorithm descriptions into runnable Python code. Secure QR Code Scanner

Designed to combat rising phishing attacks hidden in QR codes. Created both a GUI and web-based tool that detects and blocks malicious URLs using a Linear SVM classifier trained on 87 phishing-related features. Reached 97% accuracy and warned users in real time before opening unsafe links.
